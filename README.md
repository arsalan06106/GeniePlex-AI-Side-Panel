
# GeniePlex Ultimate AI Side Panel

## Overview

**GeniePlex Ultimate AI Side Panel** is a Chrome extension that provides instant access to all your favorite AI tools directly from the Chrome side panel. Boost productivity with a split-screen workflow.

[![Chrome Web Store](https://img.shields.io/badge/Chrome-Web%20Store-blue?logo=google-chrome)](https://chromewebstore.google.com/detail/kcilllhjfmefpahddhdbpfmghjkffiob?authuser=0&hl=en)

## Supported AI Services

- Gemini
- ChatGPT
- Claude
- Copilot (Microsoft)
- DeepSeek
- Grok
- Mistral
- Perplexity
- Qwen
- Kimi
- GitHub Copilot

## Features

- **Side Panel Integration:** Access AI tools instantly from Chrome's side panel for seamless multitasking
- **Split Screen Workflow:** Work with AI and web content simultaneously in split view
- **Drag-and-Drop Reordering:** Customize the order of AI service buttons to match your workflow
- **Custom Links:** Add any URL to the toolbar via the settings menu
- **Service Visibility Toggles:** Show or hide individual AI services to keep your toolbar clean
- **Keyboard Shortcuts:** Navigate between AI models without using your mouse
  - Toggle extension: `ALT + Q` (Mac: `Command + Shift + A`)
  - Next AI model: `ALT + SHIFT + N`
  - Previous AI model: `ALT + SHIFT + P`
- **Content Extractor:** Smart extraction that automatically cleans and formats text from news articles, code repositories, and social posts for AI analysis
- **Remember Last Model:** Automatically loads your last selected AI service
- **Scrollbar Always Visible:** Optional setting to keep the scrollbar always visible
- **Liquid Glass Theme:** Modern, professional dark theme design
- **Instant Loading:** No blank screen - inline critical CSS ensures instant visibility
- **Non-Blocking JavaScript:** Fast initialization that doesn't freeze the UI
- **Iframe Caching:** Fast service switching with cached iframes

## Installation

1. **Download the Extension**: Clone this repository or download the latest release from [GitHub Releases](https://github.com/arsalan06106/GeniePlex-AI-Side-Panel/releases).
2. **Load the Extension into Chrome**:

   - Open Chrome and navigate to `chrome://extensions/`.
   - Enable "Developer mode" using the toggle in the top-right corner.
   - Click on "Load unpacked" & select the `Source` directory from where you have cloned/downloaded the extension.

## Usage

1. **Open the Side Panel**:
   - Click the extension icon in the Chrome toolbar.
   - Or use the keyboard shortcut `ALT + Q` (Mac: `Command + Shift + A`) to open/close the side panel.

2. **Navigate Between AI Services**:
   - Use `ALT + SHIFT + N` to switch to the next AI model.
   - Use `ALT + SHIFT + P` to switch to the previous AI model.

3. **Customize Your Toolbar**:
   - Drag and drop AI service buttons to reorder them.
   - Add custom links via the settings menu.
   - Toggle visibility of individual services.
