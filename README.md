# GeniePlex Ultimate AI Side Panel

Access all your favorite AI tools instantly in your Chrome side panel.
Boost productivity with a split-screen workflow.

<div align="center">

GeniePlex Ultimate AI Side Panel is a Chrome extension that puts 11 AI services one click away from any tab.
It opens as a native side panel — no new windows, no context switching — so you can browse and use AI at the same time.

## Features

- **Side Panel Integration** — AI tools live inside Chrome's native side panel, always one click or shortcut away
- **Split Screen Workflow** — Run AI and your current tab side by side without resizing anything
- **Drag-and-Drop Reordering** — Arrange the toolbar to match how you actually work
- **Custom Links** — Add any URL to the toolbar through settings
- **Service Visibility Toggles** — Hide what you don't use, keep what you do
- **Content Extractor** — Pulls clean text from articles, repos, and social posts so you can feed it straight to AI
- **Remember Last Model** — Reopens with whatever AI service you used last
- **Liquid Glass Theme** — Dark interface with a polished glass aesthetic
- **Instant Loading** — Inline critical CSS means the UI appears immediately, no blank screen
- **Iframe Caching** — After first load, switching between AI services is near-instant

## Installation

### Chrome Web Store

The simplest route. Click the badge below, then hit Add to Chrome.

<div align="center">

[![Chrome](https://raw.githubusercontent.com/creosB/creosB/refs/heads/main/assets/Chrome%20Web%20Store.svg)](https://chromewebstore.google.com/detail/kcilllhjfmefpahddhdbpfmghjkffiob?authuser=0&hl=en)

</div>

### From Source

1. Clone this repo or download the latest release from [GitHub Releases](https://github.com/arsalan06106/GeniePlex-AI-Side-Panel/releases).
2. Open Chrome and go to `chrome://extensions/`.
3. Turn on **Developer mode** in the top-right corner.
4. Click **Load unpacked** and select the `Source` directory.

---

## Usage

Click the extension icon in the Chrome toolbar — or press `Alt + Q` — to open the side panel. Click any AI service icon to load it. Use `Alt + Shift + N` and `Alt + Shift + P` to cycle through services without touching the mouse.

Drag and drop the icons to reorder them. Open settings to add custom links, toggle individual services on or off, or enable split view.
